## 设置

![Auto Save](image-31.png)
```json
 "files.autoSave": "afterDelay"
```

![cursorSmoothCaretAnimation](image-32.png)
```json
"editor.cursorSmoothCaretAnimation": "on"
```
![mouseWheelZoom](image-33.png)
```json
"editor.mouseWheelZoom": true
```

![autoGuessEncoding](image-34.png)
```json
"files.autoGuessEncoding": true
```

![smoothScrolling](image-35.png)
```json
"editor.smoothScrolling": true,
"workbench.list.smoothScrolling": true,
"terminal.integrated.smoothScrolling": true
```

![formatOnPaste](image-36.png)
```json
"editor.formatOnPaste": true
```

![formatOnType](image-37.png)
```json
"editor.formatOnType": true
```

![wordWrap](image-38.png)
```json
"editor.wordWrap": "on"
```

![bracketPairs](image-39.png)
```json
"editor.guides.bracketPairs": true
```

![acceptSuggestionOnEnter](image-40.png)
```json
"editor.acceptSuggestionOnEnter": "smart"
```
![suggestSelection](image-41.png)
```json
"editor.suggestSelection": "recentlyUsed"
```
![dialogStyle](image-42.png)
```json
"window.dialogStyle": "custom"
```
![showBreakpointsInOverviewRuler](image-43.png)
```json
"debug.showBreakpointsInOverviewRuler": true,
```
![exclude](image-44.png)
```json
"files.exclude": {
  "**/node_modules": true
}
```
![linkedEditing](image-45.png)
```json
"editor.linkedEditing": true
```
